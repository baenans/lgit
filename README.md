Laser Game
==========

Laser Game is a game developed by [Furu](https://github.com/furubu) and myself, using [Polymer](https://www.polymer-project.org/1.0/), [Progressive Web Apps](https://developers.google.com/web/progressive-web-apps?hl=en) and other web technologies!.


# Install

Download the repo using 

```
git clone https://github.com/baenans/lgit.git
```
